📌 Project Overview

This project performs a Mini Exploratory Data Analysis (EDA) on the famous Titanic Dataset using Python.

The objective is to clean the dataset, analyze survival patterns, and create professional visualizations to uncover meaningful insights from the data.

This task focuses on improving data analysis, visualization, and storytelling skills using real-world data.

📂 Dataset

Dataset Used: Titanic Dataset (Kaggle)

The dataset contains passenger information such as:

Age

Gender

Passenger Class

Embarkation Port

Family Details

Survival Status

🛠 Tools & Technologies

Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook / Google Colab

📊 Tasks Performed

1. Data Cleaning
Filled missing Age values using mean imputation.
Removed irrelevant columns like Cabin.
Checked and handled missing values.

2. Exploratory Data Analysis (EDA)

Analyzed:

Survival rate by Age Group

Survival rate by Embarkation Port

Survival rate by Family Size (SibSp + Parch)

3. Data Visualization

Created:

Histogram of passenger age distribution

Heatmap of correlations

Bar plot showing survival by family size

Conclusion

This project improved my understanding of data cleaning, EDA, and data visualization using Python.
The analysis revealed that factors like gender, age, passenger class, and family size affected survival chances on the Titanic.
